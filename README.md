📱 App de Eventos (Wireframe)
📌 Descrição do Projeto

Este projeto consiste no desenvolvimento de um aplicativo mobile de eventos em formato de wireframe (baixa fidelidade), criado no Figma. O objetivo é facilitar a criação, visualização e personalização de eventos, com foco na experiência do usuário.

🎯 Objetivo

Praticar:

UX/UI Design

Prototipação no Figma

Organização de telas

Fluxo de navegação

🎨 Wireframe

O projeto foi desenvolvido como wireframe, ou seja:

Telas "simples"

Mais ou menos uso de cores

Foco na estrutura e funcionalidade

🔄 Fluxo Navegável

O protótipo possui navegação entre telas, permitindo que o usuário:

Acesse diferentes páginas

Interaja com botões

Navegue pelo aplicativo

🔀 Dois caminhos para a mesma tarefa

O app possui dois fluxos diferentes para realizar a mesma ação:

Criação de Evento Avançado (evento mais detalhado)

Criação de Evento Vapt-Vupt (evento rápido)

Ambos levam ao mesmo resultado final (evento criado).
